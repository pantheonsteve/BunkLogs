# Counselor Flow

Stories 1–9. The Counselor is the primary daily observer of a fixed bunk roster.

**Pattern:** Anywhere a single person is responsible for daily observation of a fixed roster — religious-school teacher, group home staff, daycare lead, therapist's daily notes — this flow applies.

See `../00_cross_cutting/` for the visibility model, audit trail, and i18n layers this flow references.

## Stories

1. Sign in with Google on a phone
2. Land on a dashboard that shows what I owe today
3. See which campers in my bunk still need a reflection
4. Edit a camper reflection submitted today
5. Submit my daily self-reflection
6. Edit today's self-reflection; view past reflections read-only
7. Submit a camper-care request
8. Submit a maintenance request with optional urgency and photo
9. See an unambiguous "all set" state when today's required work is complete
