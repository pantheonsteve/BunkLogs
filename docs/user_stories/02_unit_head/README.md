# Unit Head Flow

Stories 10–17. The Unit Head is a mid-level supervisor with read access to a defined scope of work and triage/pattern-detection as their primary job.

**Pattern:** Religious-school director overseeing Madrich cohorts, department head, charge nurse with unit visibility, shift supervisor across multiple work crews.

## Stories

10. Sign in and see all bunks under my supervision
11. Open a bunk dashboard and read the day in one screen
12. Read the daily score grid at a glance
13. Open a camper's full daily reflection with trend
14. See today's bunk orders without leaving the bunk view
15. Read specialist reports about campers in this bunk
16. Submit my Unit Head self-reflection
17. Edit today's UH reflection; view history read-only
